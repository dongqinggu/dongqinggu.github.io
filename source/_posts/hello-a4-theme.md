## 为什么选择 A4？

在众多的 Hexo 主题中，A4 最打动我的是它的设计理念：**让读者专注于阅读文字，写者专注于写作**。

没有花哨的动画，没有复杂的布局，就像一个干净的白纸 —— 这就是 A4。

## 搭建过程

搭建过程非常简单，只需几步：

1. 初始化 Hexo 项目
2. 安装 `hexo-theme-a4` 主题
3. 创建 `_config.a4.yml` 配置文件
4. 生成页面和文章
5. 本地预览

### 快速开始命令

```bash
hexo init blog
cd blog
npm install hexo-theme-a4@latest
hexo new page index
hexo new page list
hexo s
```

## 使用感受

A4 的 A4 纸张质感让人耳目一新。特别是：

- **极简设计**: 没有多余元素，阅读体验非常好
- **暗黑模式**: 一键切换，晚上阅读不伤眼
- **排版精致**: 中文字体渲染效果很赞
- **响应式**: 手机和桌面端都有良好体验
